# GraphPilot Java XML Sample

Java + XML 기반 GraphPilot 탐색 테스트 앱입니다.

포함 패턴:

- BottomNavigationView + Fragment 화면 전환
- WebView: 로컬 HTML 렌더링
- RecyclerView 복잡 상품 리스트
- 단순 선택 리스트: 벨소리 RadioGroup
- 리스트 + 상세: 상품 상세, 설정 상세
- Filter: 상품 필터 버튼과 필터 초기화

주요 클릭 View에는 명확한 `android:id`, `text`, `contentDescription`을 넣었습니다.
